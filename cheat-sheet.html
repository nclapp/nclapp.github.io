Ruby Arrays and Hashes: A Cheat Sheet!

By Natty Clapp and Logan Price

Create New Arrays & Hashes | Accessing Data | Obtaining Info | Adding, Altering & Removing Data

Creating New

a = Array.new
Creates an empty array named "a" (or whatever is on left of =)
a = []
Creates a new empty array
a = Array.new(3)
Creates a new array with 3 empty slots: [nil, nil, nil]
a = Array.new(3, "hi")
Creates a new array with 3 elements with default value of "hi": ["hi","hi","hi]
a = [1,2,"hello"]
Creates a new array with given elements: [1, 2, "hello"]
a = %w(hi there everyone)
Creates a new array by turning each space-separated word into a string element: ["hi", "there", "everyone"]
h = Hash.new
Creates an empty hash named "h" (or whatever is on left of =)
h = Hash.new("bird")
Creates a hash with a default value of "bird" for keys that do not exist (returns nil if no default given)
h = {}
Creates a new empty hash
h = {"age" => 34, "height" => 76}
Creates a hash with two key-value pairs: {"age" => 34, "height" => 76}
h = {age: 34, height: 76}
Creates a new hash where the keys are symbols: {:age => 34, :height => 76}

Accessing Data

a = [1,2,3,4,5,6,7,8]

a[3]
Returns the fourth (index 3) element of array "a": 4
a[-3]
Returns the third to last (third from right) element in the array, "a": 6
a[2,3]
Returns an array of 3 consecutive elements of array "a", starting at index 2: [3,4,5]
a[3..6]
Returns an array of elements indexed from 3 to 6 (a range): [4,5,6,7]
a.values_at(1,4,6)
Returns an array of elements at each specified index: [2,5,7]
h = {"a" => 1, "b" => 2, "c" => 3, "d" => 4}

h["a"]
Returns the value of key "a": 1
h.keys
Returns an array of all the keys in hash "h": ["a","b","c","d"]
To return a specific key, you can call h.keys[i] where i is the index of the key
h.values
Returns an array of all the values in hash "h": [1,2,3,4]
To return a specific value, you can call h.values[i] where i is the index of the value
h.values_at("a","c")
Returns an array of the values at each specified key: [1,3]

Obtaining Information

a = [1,2,3,4,5,6,7,8]
h = {"a" => 1, "b" => 2, "c" => 3, "d" => 4}

a.count; h.count
a.size; h.size
a.length; h.length
Returns the number of elements in the array or number of key/value pairs in a hash
a.count returns 8
h.count returns 4
a.empty?; h.empty?
Returns a boolean value stating whether the array or hash is empty
a.include?(2); h.include?("a")
Returns a boolean value stating whether the array includes the argument passed or a hash the key of the argument passed
(For a hash, you can do the same thing with h.has_key?("a"))
h.has_value?(2)
Similar to has_key. Returns boolean stating whether the given argument exists as a value in the hash

Adding, Removing, and Altering Items

a = [1,2,3,4,5,6,7,8]
h = {"a" => 1, "b" => 2, "c" => 3, "d" => 4}

a.push(9)
Inserts the argument at the end of the array: a now equals [1,2,3,4,5,6,7,8,9]
a[15] = "bird"
Sets the element at index 15 to "bird"
If "a" has fewer elements than index 15, it will pad the array with nil or the default value until a[15], which will be "bird"
(a now equals [1, 2, 3, 4, 5, 6, 7, 8, 9, nil, nil, nil, nil, nil, nil, "bird"])
h["e"] = 5
Sets the key of "e" to the value 5. If key "e" does not exist, it will create the key/value pair of "e" => 5
a.unshift("cat")
Inserts argument(s) "cat" (comma-separate multiple arguments) at the beginning of the array
[1,2,3,4] becomes ["cat",1,2,3,4]
a + ["boop","beep"]
Adds elements of a second array to the end of the first array (concatenates)
a.pop; a.pop(2)
Removes the last item of an array and returns the item. If integer argument is passed, it removes that number of items from the end of the array and returns them as an array
a.pop: [1,2,3,4] becomes [1,2,3] and returns 4
a.pop(2): [1,2,3,4] becomes [1,2] and returns [3,4]
a.delete_at(3)
Removes the item at index 3
a.shift; a.shift(2); h.shift
Removes the first item of an array (if argument passed, removes that number of items).
Removes the first key/value pair of a hash
